# 📊 Sales Data Analysis Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive **Sales Data Analysis Dashboard** built using **Power BI**.
The goal of the dashboard is to analyze sales performance across different regions, categories, and time periods to generate meaningful business insights.

The dashboard allows users to easily explore trends, compare performance, and make data-driven decisions.

---

## 🎯 Objectives

* Analyze overall sales performance.
* Identify sales trends over time.
* Compare sales across regions and product categories.
* Highlight top-performing products and areas.

---

## 🛠 Tools & Technologies

* **Power BI Desktop**
* **Microsoft Excel / CSV Dataset**
* **Power Query Editor** (Data Cleaning)
* **DAX (Data Analysis Expressions)**

---

## 📂 Dataset

The dataset used in this project contains retail sales transaction data with the following fields:

* Order ID
* Order Date
* Ship Date
* Customer Name
* Segment
* Country
* City
* State
* Region
* Category
* Sub-Category
* Product Name
* Sales

The dataset includes **thousands of sales records** across multiple regions and product categories.

---

## 🧹 Data Cleaning & Transformation

Data preprocessing was performed using **Power Query Editor**:

* Converted **Order Date** and **Ship Date** to Date format
* Trimmed extra spaces from text fields
* Removed unnecessary columns
* Corrected data types
* Prepared fields for analysis

---

## 📊 Dashboard Features

### KPI Cards
